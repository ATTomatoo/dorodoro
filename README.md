Dorodoro 文字冒险游戏插件
NoneBot Plugin
Version

Dorodoro 是一个基于 NoneBot2 框架的文字冒险游戏插件，提供交互式故事体验。

功能特性
交互式文字冒险游戏体验

支持分支剧情选择

可配合图片展示增强游戏体验

多用户独立游戏进度保存

安装方法
确保已安装 NoneBot2 框架

使用 pip 安装插件：

bash
pip install nonebot-plugin-dorodoro
在 NoneBot2 项目中加载插件

使用方法
开始游戏
/doro 
做出选择
/choose <选项> 或 /选择 <选项>
